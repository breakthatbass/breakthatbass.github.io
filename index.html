Projects

xmas

A cli program to download puzzle input files and submit puzzle answers for Advent of Code puzzles. It tries to cache everything it can to minimize calls to the AoC servers.

tinyfs

A simple server/client program in C to share files of any type and any size between systems in a LAN.

c-libs

A collection of libraries I've built over time in C. There are a bunch of string processing functions, a list library, a super simple header file timing library, and a bunch of other pretty cool stuff. This repo is always growing.

conway

A visualizer for John Conway's game of life algorithm in C. It reads from run length encoded (RLE) files, which are compressed files containing data on placement of squares in a grid. The program reads in an RLE file, decompresses it, and runs the algorithm over the grid. And you get to see it happen!

Tools

cdox

A tool to create documentation of C source files in Markdown format based on comments in the code. It's sort of like Javadocs or Doxygen except much simpler and only has a small amount of keywords.

jvm-algs4

A program to automate the compiling and running of Java programs for use with the textbook Algorithms by Robert Sedgewick and Kevin Wayne. For most of the exercises in the book, you have to use the library provided with the book. jvm-algs4 lets you compile and run the code while more easily linking the library.

conv

A tool to add and convert decimal, hexadecimal, and binary numbers. It takes any number of arguments and attempts to add them together. It gives the sum in three forms: decimal, hexadecimal, and binary. It's a tool mostly for working with embedded systems.

k&r

Most computer science textbook are so freaking huge, I have doubts at the number of people that make it though the entirety of these books. Adult life just simply doesn't provide enough time for such endeavors. The C Programming Language aka k&r is an anomaly in CS textbooks. Not counting the appendices, it comes in at only 189 pages and covers the entirety of the C language plus plenty of systems level POSIX stuff. It's really a masterpiece in technical writing. This repo contains all my solutions to the exercises in the book. If you don't have this book, go buy it now.

cs50

Harvard provides a free online course called cs50 which acts as an introduction to computer science. It covers all the basics of CS without going to deep into any of it. Most of the course uses C and the rest Python. This course set me up pretty well. After, I stuck with both C and Python and still heavily use both languages to this day. This repo holds all my work from the course.
