Projects

xmas

A cli program to download puzzle input files and submit puzzle answers for Advent of Code puzzles. It tries to cache everything it can to minimize calls to the AoC servers. Uses the libcurl C library.

tinyfs

A simple server/client program in C to share files of any type and any size between systems in a LAN.

c-libs

A collection of libraries I've built over time in C. There are a bunch of string processing functions, a list library, a super simple header file timing library, and a bunch of other pretty cool stuff. This repo is always growing.

conway

A visualizer for Conway's game of life algorithm in C. It reads from run length encoded (RLE) files, which are compressed files containing data on the placement of squares in a grid.

Tools

cdox

A tool to create documentation of C source files in Markdown format based on comments in the code. It's sort of like Javadocs or Doxygen except much simpler.

jvm-algs4

A program to automate the compiling and running of Java programs with the algs4 lib for use with the textbook Algorithms by Robert Sedgewick and Kevin Wayne.

conv

A tool to add and convert decimal, hexadecimal, and binary numbers. It takes any number of arguments and attempts to add them together. It gives the sum in three forms: decimal, hexadecimal, and binary.
